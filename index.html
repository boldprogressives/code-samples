Checkboxes

Take a look at a standard radio/checkbox code. You can break it down into three sections.

<input type="checkbox" name="action_loves_ice_cream" value="true" id="ice-cream" />
<label for="ice-cream">I love ice cream</label>

<input type="checkbox" name="action_loves_pie" value="true" id="pie" />
<label for="ice-cream">I love pie</label>

The "name" and the "value" are the parts of our code that get recorded into our database. Think of "name" as the question you're asking the member and "value" as the member's answer to the question. When you use checkboxes, a member can click as many as they want, and as long as they all have different "names" or different "values", they'll all be recorded safely in the database.

To learn more about choosing a name, be sure to read the ActionKit/ActionFields Guidelines. (for example, names should ALWAYS start with the prefix action_; that's what tells the database it's time to start storing information.)

Radio Buttons

For questions where you want the member to only choose one answer ("Do you want ice cream? Yes, No, Maybe"), you should use radio inputs with the same name. This ensures that a user can only select one option at a time. (But the different inputs still need distinct values, or else we won't know which one they picked!)

<input type="radio" name="action_syria_usaction" value="yes" id="usaction_yes" />
<label for="usaction_yes">Yes</label>

Note for both checkboxes and radio buttons, the id and label for attributes make it easier for our members to click on the different options. For easy setup, just squish together your "name" and "value" and make sure they're the same for "id' and "label for". So if you have <input name="great" value="job" />, then id and label for would be great_job.

Hidden option

To create a hidden option, include the onClick in the <label> section of your code. Make sure that it matches to a <div></div> where you will put the dropdown content. See below for code.

<input type="radio" name="action_icecream" value="na" id="icecream_na" onClick="$('.hiddenoption_name_goes_here-div').toggle(500);"/>
<label for="icecream_na" >I don't know.</label>

<div class="hiddenoption_name_goes_here-div" style="display:none; /">
Put your hidden option content here.
</div>
